How to add Images and Videos

This section will focus on the different ways that we can add images or videos to our website. There are multiple ways to add a video or image to your page. First we will show the img tag in use. Next up will be the figure tag with the figcaption included. Finally we will go over the video tag along with some of its attributes. Let's get started

Steps to adding an image using img tag:

  1. Add img tag. Remember this is a self closing tag.
  2. Add the src attribute and add the URL of the image you would like to link. This can also be done using a relative path.
  3. Next add the alt attribute. This technically does not need to be added but it helps with screen readers and the SEO of the page. You can add a description of the picture.
  4. Next close it out with />

Steps to adding an image using figure tag with figcaption:

  1. First you want to add your figure tag.
  2. Next you're going to add your img tag and use the steps above.
    Note: Other attributes you can add too img are width/height. Or you can later style these elements in CSS.
  3. Now you can add your figcaption opening tag.
  4. Go ahead and add a caption in and it will appear at the bottom left of the image you want to display.
  5. Finally go ahead and close out both tags. Make sure you do them in order figcaption then figure.

Steps for adding a video using video tag:

  1. The first step you want to take is add your opening video tag.
  2. Now you can add your src attribute along with the URL/relative path to the video you would like to display.
    Note: with video like image you can add the follwing attributes width/height or you can style them in CSS.
  3. After you add your URL add the controls attribute. This will the the browser to add default pause, play, or fullscreen buttons to your video.
  4. Close your opening tag and its common practice to add "Video Not Supported" just in case the browser doesn't load your video this will be displayed.

Common CSS Selectors

  • Type - this is basically just selecting all of the type of element. For example if you write a CSS ruleset for type h2, the following style would style all h2 elements.

    h2 {
    color: red;
    }
  • Class - This is probably the most common and customizable selector. You use the class attribute in your HTML file, and then you can select to style that class like so (.class). Make sure to put a period in front of the class you would like to style.

    .class {
    background-color: #14bb12;
    }
  • Universal - This slector should only be used when absolutely nessicary. The will still all the elements at once.

    * {
    box-sizing: border-box;
    }
  • id - With this selector it is important to style one element at a time. These are for more unique styles for elements. This meaning that it can not be recycled like the class attribute and should be used when needed. Put the hash (#) in front to select your id in CSS.

    #id {
    font-family: 'Zen Dots';
    font-weight: bold;
    }
